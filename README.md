# my-tool

日々の作業をシンプルにするためのツールです。

## 概要

my-tool は、直感的な操作ですぐに使いはじめられる軽量ツールです。  
HTML + CSS のみで構成されており、依存関係なしで動作します。

## 機能

- **高速** — 無駄を省いたシンプルな設計で、すばやく動作
- **シンプル** — 複雑な設定不要、インストール後すぐに使える
- **安心** — データはローカルで管理、プライバシーを保護

## 使い方

```bash
git clone https://github.com/toruyumi0904-hash/my-tool.git
cd my-tool
```

`index.html` をブラウザで開くだけで動作します。

## ファイル構成

```
my-tool/
├── index.html   # ランディングページ
└── style.css    # スタイルシート
```
